# モデルの配置場所

このディレクトリにクライアントへ配信するモデルファイルを配置してください。

## 推奨構成

```text
public/models/
├── [model_name]/
│   ├── config.json
│   ├── tokenizer.json
│   ├── model.onnx (または .bin など)
│   └── ...
```

## 注意事項
- 巨大なモデルファイル（.onnx, .bin など）は `.gitignore` により Git の管理対象から除外されています。
- リポジトリにはディレクトリ構造を維持するための `.gitkeep` とこの `README.md` のみが含まれます。
- 実際にアプリケーションを動作させるには、適切なモデルファイルを手動、またはダウンロードスクリプトなどで配備する必要があります。
